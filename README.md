监控器部分：
HostServer  ：主服务器，负责与客户端传输数据，与从服务器传输数据
SlaveServer ： 从服务器，负责与主服务器交互，与本地的流量转发器接收访问数据
Client_monitor ： 客户端，负责提供从主服务器获取到的和本地的一些数据，供算法调用

公共函数模块：
IOfunc ： 主要是关于socket的函数包装


监控器每个模块初始化里都有一个debug的线程，供输出信息，可注释掉，也可用来了解数据结构

注意，程序中用到系统自带的ping，如果不是Windows中文版可能有问题

比较难看懂的数据结构是有关status的，其中一部分为：[状态，访问次数，访问持续时间，开始时间]
